# `@plinth-ui/menu-button`

[WAI-ARIA](https://www.w3.org/TR/wai-aria-practices-1.2/#menubutton)

## Installation

```
npm install @plinth-ui/menu-button
# or
yarn add @plinth-ui/menu-button
```

### Usage

```
import Alert from '@plinth-ui/menu-button'
```
